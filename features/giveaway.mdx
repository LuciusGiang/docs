### **Starting a giveaway**

<CodeGroup>

```javascript Syntax
/giveaway start <prize> <winners> <duration> [channel]
```


```javascript Example 1
/giveaway start 1 month nicho 1 1w
```


```javascript Example 2
/giveaway start 1 month nicho 1 1w #giveaway
```

</CodeGroup>

### Ending or rerolling giveaways

You can end a giveaway early or reroll the winners of a giveaway that has ended.

If you need help with getting the Message ID, visit [this page](/resources/message_id).

<CodeGroup>

```javascript Syntax
/giveaway end <message_id>
/giveaway reroll <message_id>
```


```javascript Example
/giveaway end 1346616792751149077
/giveaway reroll 1567991258432115349
```

</CodeGroup>
